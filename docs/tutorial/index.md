# Tutoriels

Bienvenue dans la section tutoriels. Vous trouverez ici des exercices pratiques pour appliquer les concepts vus en cours.

## Objectifs

- Mettre en pratique les méthodes d'estimation
- Développer l'intuition statistique
- Apprendre à analyser des données réelles

## Structure

Chaque tutoriel comprend :

1. **Énoncé** : présentation du problème
2. **Données** : description des données à analyser
3. **Questions** : série de questions guidées
4. **Indices** : aide progressive
5. **Solution** : solution détaillée avec code

## Prérequis

Pour suivre ces tutoriels, il est recommandé de connaître :

- Les bases de Python ou R
- Les concepts vus dans les chapitres de cours
